# KotiX

![alt text](kotix.png)
Kotix is a custom AI-powered with LLM support distributive for development.

https://github.com/ollama/ollama

To run and chat with Llama 3.1:
`ollama run llama3.1`

Based on Debian 12
[Main page](https://nolfgk.github.io/KotiX/)
